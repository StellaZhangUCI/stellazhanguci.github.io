Prize structure shift

  • Grand Prize size. $15,000 (2017 School-of-Medicine award) → $25,000 (2020, pandemic-era peak) → $20,000 (2022–2025 standard).
  • Standard track payouts. $4,000 / $1,500 in 2017 → $10,000 / $5,000 from 2018 onward, roughly a 6× jump.
  • MIE Award. Introduced 2020 (Baby Box first MIE-affiliated winner). Recurring specialty since.
  • Audience Favorite. Intermittent (2020, 2021, 2025); typically $1,000.
